Sobre

O Jogo da Forca é um clássico jogo de adivinhação de palavras que tem entretido gerações. Neste projeto, desenvolvemos uma versão interativa e moderna do jogo, projetada para proporcionar uma experiência divertida e educativa aos usuários.
Este projeto foi desenvolvido como parte do programa ADA Santander CODERS, uma iniciativa que visa capacitar novos talentos na área de tecnologia. A equipe responsável pela criação deste jogo é composta por Gabryel Costa, Pedro, Maria Eduarda e Jefferson Tavares. Juntos, eles uniram suas habilidades e conhecimentos para entregar um produto de alta qualidade.
Tecnologias Utilizadas Para o desenvolvimento deste projeto, foram utilizadas as seguintes tecnologias:
HTML: Estruturação do conteúdo e elementos do jogo.
CSS: Estilização e design, garantindo uma interface atraente e intuitiva.
JavaScript: Lógica do jogo e interatividade, permitindo a manipulação dinâmica dos elementos.
Manipulação do DOM: Atualização e controle dos elementos da página em tempo real, proporcionando uma experiência fluida e responsiva.

Funcionalidades do Jogo

O jogo permite que os usuários:
Joguem: Adivinhem palavras, letra por letra, com um teclado virtual.
Chutem: Tentem adivinhar a palavra inteira de uma vez. Reiniciem: Comecem um novo jogo a qualquer momento. Recebam Dicas: Utilizem dicas/temas para facilitar a adivinhação.
Além disso, o jogo utiliza palavras obtidas de uma API externa, garantindo uma variedade de desafios e mantendo o jogo sempre interessante.
Conclusão
Este projeto não só demonstra a capacidade técnica da equipe em utilizar tecnologias web modernas, mas também reflete o espírito colaborativo e a dedicação dos desenvolvedores. O resultado é um jogo envolvente e educativo, que certamente proporcionará muitas horas de diversão aos seus usuários.
